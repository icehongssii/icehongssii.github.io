중복 삭제 sql 쿼리(join이용)

마지막수정 2024-11-29 11:17:47 - 최초작성2024-11-29 11:17
태그 sql
tl;dr

👯‍♂️ 개요 및 결론

DELETE kco FROM adoc_v1.kindoc_care_opinions kco
LEFT JOIN ( SELECT MIN(id) AS keep_id FROM adoc_v1.kindoc_care_opinions
            GROUP BY center_id, platform_id) AS keep_ids
ON kco.id = keep_ids.keep_id
WHERE keep_ids.keep_id IS NULL;
  • kindoc_care_opinions 에서 centerid, platformid가 모두 중복인 객체를 삭제하고 싶음
  • 서브쿼리 SELECT MIN(id) AS keep_id FROM adoc_v1.kindoc_care_opinion GROUP BY center_id, platform_id 에서 고유한 id 를 먼저 추출하고
  • 전체 kindoc_care_opinion에서 매칭되지 않은 id는 제외하고 삭제 ✍️✍️✍️✍️✍️✍️✍️✍️ 중복되는 것들이므로
